Sand_timer_1

Цифровий пісочний годинник на Arduino

Папки репозиторію

- 3dprint — моделі корпусу для друку
- firmware — код для Arduino
- libraries — бібліотеки (можна встановити через IDE)
- schemes — схеми підключення
- test — моделі піску для тестування алгоритмів

Для підключення звуку

1.  Додайте мінідинамік (бажано з підсилювачем на транзисторі) на PIN D9.
2.  Розкоментуйте рядок `#define BUZZER_PIN 9`.
3.  У налаштуваннях годинника виберіть одну з мелодій та/або відрегулюйте гучність.

Керування годинником

- натискання першої кнопки — зменшити час на 1 сек
- натискання та утримання першої кнопки — зменшити час на 10 сек
- натискання другої кнопки — збільшити час на 1 сек
- натискання та утримання другої кнопки — збільшити час на 10 сек
- натискання двох кнопок — скидання таймера
- утримання двох кнопок — вхід у меню
- перехід між пунктами меню — утримання однієї з кнопок

Меню циклічне. Зверху — іконка пункту, знизу — значення. Меню пам'ятає останній використаний пункт протягом 30 сек. Меню вимикається через 5 сек бездіяльності.

Енергоспоживання

- при максимальній яскравості — 200 мА
- при яскравості 2 — 60 мА, під час відтворення звуку до 72 мА
- при мінімальній яскравості — 35 мА, під час відтворення звуку до 45 мА
